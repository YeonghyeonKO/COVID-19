#### 2. nls 모델 피팅 ####

Gomp, Logit, Bert 세 가지 모델 적합하는 일입니다. 학생들의 과제 제출 내역을 받기는 했는데, 정리된 테이블이 아니라 개별 R code로 받아서 
1) a는 주어진 데이터의 최종 날짜 기준으로 2) b, c는 기존 피팅했던 자료의 범위 내에서 initial value를 잡고 batch로 일단 돌려볼 예정입니다.
(e.g. 기존 Logistic에서 전세계 모델 적합 결과 3.807 < b < 15.61548이므로 initial value를 두 값 사이에서 선택) 

- 데이터 전처리가 완료된 후 바로 진행할 수 있도록 이도은 선생님이 전체 나라에 대해 한 번에 돌 수 있게 코드 작성해둘 예정입니다.
- 위의 1차 모델 피팅이 끝나는 대로 주어진 값들로 정상 피팅이 잘 안 되는 모델들을 저희 네 명이 각각 나눠서 피팅하도록 하겠습니다.
